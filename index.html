Solving a System of Differential Equations

Complex Eigenvalues

The Problem

x' = Ax,   x(0) = [1, 0]T
A =   3    −2 4    −1  

Find the solution to the initial value problem without using complex numbers

The Eigenvalue Method

When the eigenvalues are complex λ = α ± βi, we find a complex eigenvector and take its real and imaginary parts to obtain real solutions.

We find the eigenvalues of matrix A by solving |λI − A| = 0

Step 1 Solution

The characteristic polynomial:
|λI − A| = (λ−3)(λ+1) − (−2)(4) = λ² − 2λ + 5
Solving the quadratic equation:
λ = (2 ± √(4−20))/2 = (2 ± √(−16))/2 = (2 ± 4i)/2

The Eigenvalues

λ = 1 ± 2i
Step 2: Finding an Eigenvector
We find an eigenvector v for λ = 1 + 2i

(−2+2i)v₁ + 2v₂ = 0

Step 2 Solution

The matrix (λI − A) for λ = 1+2i:
[(−2+2i), 2; −4, (2+2i)]
From the first equation:
(−2+2i)v₁ + 2v₂ = 0 → v₂ = (1−i)v₁
Choosing v₁ = 1:
v₂ = 1 − i

The Eigenvector

v = [1, 1−i]T

Substituting t = 0:
[1; 0] = C₁[cos0; cos0+sin0] + C₂[sin0; sin0−cos0]
Simplification:
[1; 0] = C₁[1; 1] + C₂[0; −1] = [C₁; C₁ − C₂]
From the first equation: C₁ = 1
From the second equation: C₁ − C₂ = 0 → C₂ = 1

Initial Value Problem Solution

x(t) = et(cos2t + sin2t)
y(t) = 2etsin2t

Step 1 Solution - Finding Eigenvalues

The characteristic polynomial:
|λI − A| = (λ−3)(λ+1) − (−2)(4) = λ² − 2λ + 5
Solving the quadratic equation:
λ = (2 ± √(4−20))/2 = (2 ± √(−16))/2 = (2 ± 4i)/2

The Eigenvalues

λ = 1 ± 2i

Step 2 Solution - Finding an Eigenvector

The matrix (λI − A) for λ = 1+2i:
λI − A = [(1+2i)−3, −(−2); −4, (1+2i)−(−1)] = [−2+2i, 2; −4, 2+2i]
The first equation (λI − A)v = 0:
(−2+2i)v₁ + 2v₂ = 0
Solving for v₂:
v₂ = −(−2+2i)v₁/2 = (2−2i)v₁/2 = (1−i)v₁
Choosing v₁ = 1:
v₂ = 1 − i

The Eigenvector

v = [1, 1−i]T

Step 3 Solution - The General Solution (Full Detail)

Step a: The complex solution
x(t) = eλtv = e(1+2i)t[1; 1−i]
Step b: Decomposing the exponential
e(1+2i)t = et · e2it
Step c: Euler's formula
e2it = cos(2t) + i·sin(2t)
Step d: Multiplying by the eigenvector
et(cos2t + i·sin2t) · [1; 1−i]
Step e: Computing the first component x(t):
x(t) = et(cos2t + i·sin2t) · 1 = et(cos2t + i·sin2t)
Real part: etcos2t   |   Imaginary part: etsin2t
Step f: Computing the second component y(t):
y(t) = et(cos2t + i·sin2t) · (1−i)

= et[(cos2t)(1) + (cos2t)(−i) + (i·sin2t)(1) + (i·sin2t)(−i)]

= et[cos2t − i·cos2t + i·sin2t + sin2t]

= et[(cos2t + sin2t) + i(sin2t − cos2t)]

Real part: et(cos2t + sin2t)   |   Imaginary part: et(sin2t − cos2t)
Step g: The two linearly independent solutions:
Solution 1 (real part): x₁(t) = et[cos2t; cos2t + sin2t]
Solution 2 (imaginary part): x₂(t) = et[sin2t; sin2t − cos2t]

The General Solution

x(t) = C₁et[cos2t; cos2t+sin2t] + C₂et[sin2t; sin2t−cos2t]

Step 4 Solution - Initial Conditions

Substituting t = 0 in the general solution:
x(0) = C₁e⁰[cos0; cos0+sin0] + C₂e⁰[sin0; sin0−cos0]
Simplification (cos0=1, sin0=0):
x(0) = C₁[1; 1] + C₂[0; −1] = [C₁; C₁ − C₂]
Comparing to the initial condition x(0) = [1, 0]T:
Equation 1:   C₁ = 1
Equation 2:   C₁ − C₂ = 0   →   C₂ = C₁ = 1

The Constants

C₁ = 1,   C₂ = 1

The Final Solution

x(t) = et(cos2t + sin2t)
y(t) = 2etsin2t
